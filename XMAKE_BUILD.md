## XMake Installation


#### via curl

```bash
bash <(curl -fsSL https://xmake.io/shget.text)
```

#### via wget

```bash
bash <(wget https://xmake.io/shget.text -O -)
```

## Build

```bash
xmake 
```

## Output 

`${ImHex}/build/macosx/x86_64/release/ImHex`
